# FREEPOWER Supply - React Project Showcase App

## Overview
FREEPOWER Supply is a React-based single-page application (SPA) that showcases a dynamic product storefront with full CRUD functionality.

This project demonstrates advanced React concepts including:

- Custom hooks
- Client-side routing
- State management
- Controlled forms
- Data fetching
- Dynamic filtering
- Component testing

---

## Features

- View products
- Search products dynamically
- Add new products
- Update product price
- Delete products
- Responsive layout
- Full test coverage for major features

---

## Technologies Used

- React 18
- React Router
- Vite
- JSON Server (mock backend)
- Vitest
- React Testing Library
